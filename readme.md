# Thermos App

Thermos is a simple social bookmarking site. Or actually, it is a demo for the Pluralsight course titled `Flask Fundamentals`.

### Stack:
1. python2.7 + Flask
2. sqlite
3. http://www.initializr.com/ to generate template

### Flask extention
1. Flask-SQLAlchemy
2. Flask-Script
3. Flask-WTF
4. Flask-Login
5. Flask-Migrate
6. Flask-Moment
7. Flask-DebugToolbar
